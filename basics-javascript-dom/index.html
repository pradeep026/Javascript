Introduction to Javascript and DOM

  • Javascript is a client-side scripting language.

  • Javascript is an object oriented scripting language.

  • Whereas HTML defines a webpage's structure and content and CSS sets the formatting and appearance,
    JavaScript adds interactivity to a webpage and creates rich web applications.

  • The HTML page is converted into a tree strctured Objects in DOM (Document Object Model).

  • Javascript interacts with DOM (DOM provides In-build API) to manipulate a web page.

How Javascript Interacts DOM ?

DOM provides an object called document to manipulate a web page.

We can modify the content of a DIV or any HTML element through this document object.

I have listed a few properites of document object.

  • document.body - Which returns the entier body of the html page.

  • document.body.appendChild - Which appends a new HTML element to the page.

  • document.title - We can get the title of the current page and as well as we can set new title to the page.

  • document.getElementById - Which returns the HTML element of the given ID.

  • document.getElementsByTagName - Which returns all the HTML element of the given tag name. It returns an array of HTMLElement.

  • document.getElementsByClassName - Which returns all the HTML element of the given css class name. It returns an array of HTMLElement.

  • document.createElement - Which helps to create HTML element dynamically in run time.
