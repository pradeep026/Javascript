Javascript variables and functions

Unlike Java and other programing language, Javascript uses `var` keyword to declare variables for all data type

Example:
var x = 'foo'; variable `x` is declared and its data type is `String`. || typeof x //"string"
var x = 1234; variable `x` is declared and its data type is `Number`. || typeof x //"number"
var x = true; variable `x` is declared and its data type is `Boolean`. || typeof x //"boolean"

Function declaration

`functionName` can be anything and it is a reference to the function.

`functionName` can contain [a-zA-Z], _ and [0-9].

`functionName` can not start with a numeric value.

N number of arguments can be passed into a function.

Syntax: 1 - This method is syntactically a statement
function functionName(){
   //body of the function
}

Syntax: 2 This method is called as function expressions
var functionName = function(){
   //body of the function
}
